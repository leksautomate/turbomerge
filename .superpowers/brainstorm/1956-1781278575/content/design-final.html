Set once per project. All jobs inherit it. Works for both images and videos.

How each provider handles aspect ratio
Runware (openai:gpt-image@2)
aspectRatio "1:1" → width: 1024, height: 1024
aspectRatio "16:9" → width: 1536, height: 864
aspectRatio "9:16" → width: 864,  height: 1536

Vertex Image (gemini-3.1-flash-image-preview / gemini-3-pro-image-preview)
imageConfig.aspectRatio = "1:1" | "16:9" | "9:16"
endpoint: aiplatform.googleapis.com/v1/publishers/google/models/:model:streamGenerateContent

Vertex Video (veo-3.1-lite-generate-001)
config.aspect_ratio = "1:1" | "16:9" | "9:16"
config.duration_seconds = 8, config.resolution = "720p" | "1080p"
client.models.generate_videos() → operation → poll until done
