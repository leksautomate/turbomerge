That's a good idea. For a serious bulk generator, everything should live inside Projects, not directly under the user.

Before generating anything, the user must create a project.

Example:

Images

Project Settings

Every project can store:

{
  "model": "imagen-4",
  "mediaType": "image",
  "aspectRatio": "16:9",
  "format": "png",
  "negativePrompt": "",
  "quality": "high"
}

For videos:

{
  "model": "veo",
  "duration": 8,
  "resolution": "1080p",
  "aspectRatio": "9:16"
}

All jobs automatically inherit these settings.
